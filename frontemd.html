Qué es Frontend?

Es la parte visual y interactiva de una página web con la que los usuarios interactúan directamente

Caracteristicas de Frontend

Incluye responsividad, interacción dinámica y diseño visual, todo en el navegador del usuario.

Como funciona Frontend

Utiliza lenguajes como HTML, CSS y JavaScript para construir la estructura, estilos y comportamientos de una página web.
